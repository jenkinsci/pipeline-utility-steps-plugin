Reads a SimpleTemplateEngine source file in the current working directory or a String as a SimpleTemplateEngine source and binds a map of key value pairs to it. The returned object is a String where template placeholders have been expanded to their values.

Example:

def binding = [
     firstname : "Grace",
     lastname  : "Hopper",
     accepted  : true,
     title     : 'Groovy for COBOL programmers'
 ]
String text = '''
Dear <%= firstname %> $lastname,

 We <% if (accepted) print 'are pleased' else print 'regret' %>
 to inform you that your paper entitled
 '$title' was ${ accepted ? 'accepted' : 'rejected' }.

 The conference committee.
'''
String desiredOutput = simpleTemplateEngine templateSource: templateSource, binding: binding
